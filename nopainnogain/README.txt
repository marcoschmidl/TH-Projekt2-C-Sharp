Guten Tag Herr Hüttl, 

Das ist die ReadMe-Date unseres Projekts.
Anbei finden Sie alle relevanten Informationen die Sie benötigen,
um unser Projekt auszuführen. 

1.Test ausführen 
Führen Sie alle Tests einmal aus, um die Datenbank zu initialisieren. 
Sofern das Program alle Tests ausgeführt hat, was 1-2 Minuten dauern könnte,
steht dieses für ihre Nutzung zur Verfügung. Danach erhalten Sie durch den Demo-Test fertige 
Daten mit denen Sie arbeiten können.

2.Sign Up 
Dort können Sie sich einen Account erstellen, welchen Sie mit all Ihren relevanten Daten befüllen. 

3.Home
Im HomeMenu kann man auf die Verschiedenen Hauptfunktionen der Anwendung zugreifen und sich ausloggen.
Zusätzlich zeigt das Program den persönlichen Calorienbedarf welcher konsumiert werden muss und 
gibt Ihnen ein Update wie viel Sie am aktuellen Tag im Überschuss oder Defizit sind.

3.Edit Profil
Wenn Sie änderungen an Ihrem Profil tätigen wollen oder die angegeben Daten sehen wollen, können Sie das 
über die Edit Profil Taste. 

4.Weight
Um den Gewichtsverlauf bestmöglich zu verfolgen und zu sehen, ob Sie ihrem Ziel näher kommen, haben Sie 
die Möglichkeit Ihr aktuelles Gewicht manuel einzutragen. Die eingetragenen Gewichte spiegeln sich dann 
im Report wieder. 

5.Tracker 
Möchten Sie ein neues Produkt erstellen?
Tragen Sie die Werte Ihres Produkts in die vorgesehenen Felder und Fügen Sie eine Menge hinzu.
Mit Add fügen Sie das Produkt der Datenbank hinzu.
Mit Consum fügen Sie das Produkt zur täglichen Konsumliste hinzu.

Möchten Sie ein vorhandenes Produkt verwenden?
Wählen Sie im Dropdown-Menü der Suchleiste Ihr gewünschtes Produkt aus, drücken Sie "Show" und fügen Sie eine Menge hinzu.
Anschließend können Sie das Produkt mit "Consum" zur täglichen Konsumliste hinzu fügen und diese einsehen.

6.Report anzeigen 
Im Report Menu haben Sie die Möglichkeit Ihren Konsum der letzten Woche zu sehen. Dort inbegriffen finden Sie die konsumierten Kalrorien
und das Gewicht welches Sie manuel eingetragen haben inklusive dem Datum des Tages. Diese Funktion ist Rückblickend für eine Woche verfügbar
um den Verlauf der Woche einzusehen.

Wir wünschen Ihnen eine schöne Userexperience mit unserem Program!
